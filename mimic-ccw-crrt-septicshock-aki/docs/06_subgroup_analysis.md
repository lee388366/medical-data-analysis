# Subgroup analysis plan (pre-specified)

All subgroup variables are defined at baseline using window (t0−6h, t0], to avoid post-treatment bias.

## Primary subgroup variables

### 1) Oliguria (3 groups; ml/kg/h)
- oliguria_<0.3: uo < 0.3
- oliguria_0.3-0.5: 0.3 ≤ uo < 0.5
- non_oliguric_>=0.5: uo ≥ 0.5

Definition:
- uo_mlkgph_t0_last6h = (SUM urine output in (t0−6h, t0]) / weight_kg / 6

### 2) Shock severity via norepinephrine-equivalent dose (3 groups; µg/kg/min)
- NE_<0.1
- NE_0.1-0.3
- NE_>=0.3

NE-equivalent unit is reported as µg/kg/min (mcg/kg/min).

### 3) Mechanical ventilation (2 groups)
- MV vs no_MV
Definition: ventilation overlap in (t0−6h, t0].

### 4) Lactate (2 groups; mmol/L)
- lactate_>=4 vs lactate_<4

### 5) Oxygenation (PF ratio; 3 groups)
- PF_<100
- PF_100-200
- PF_>=200

## Exploratory subgroup variable

### AKI pathway at t0
Using KDIGO stage components around t0:
- uo_driven: stage_uo=3 and stage_creat<3
- scr_driven: stage_creat=3 and stage_uo<3
- mixed: both stage_uo=3 and stage_creat=3
- unknown: missing / not classifiable

## Statistical reporting
- For each subgroup variable, report treatment effect within each level and the interaction p-value:
  - model includes treatment × subgroup interaction term
- Keep interpretation cautious (multiple comparisons); subgroups are hypothesis-generating unless otherwise specified.
