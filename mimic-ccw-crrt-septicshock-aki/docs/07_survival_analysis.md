# Survival analysis

## Data sources

Survival analyses are performed in R using:

- 08 clone–censor dataset
- 07 outcomes table
- t0_time and deathtime

## Analysis population

Pseudo-population generated by clone–censor–weight procedure.

Each patient contributes two clones:

- Early CRRT (≤24h)
- No early CRRT

## Primary survival analysis

Weighted pooled logistic regression will be used to estimate
discrete-time hazards.

From the fitted model we derive:

- weighted survival curves
- cumulative incidence
- risk difference at 28 days
- restricted mean time lost (RMTL)

τ = 28 days.

## Supplementary analyses

- weighted Cox proportional hazards model
- restricted mean survival time (RMST)

## Software

R packages:

- survival
- survminer
